Minecraft Enhancement Hacker


Install
-------

    1.  chmod 700 enhancement_hacker.py
    2.  wget https://raw.github.com/codewarrior0/pymclevel/master/nbt.py
    3.  The NBT library requires Numpy. Get it here: http://new.scipy.org/download.html


Usage
-----

    1.  Backup your level.dat or <PLAYERNAME>.dat files first as the may get corrupt.
    2.  /enhancement_hacker.py <level.dat> <xp level>
